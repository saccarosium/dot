# Arrays

```c
  /* scrivendo in questo modo inizializzi l'array "v", allocandoli 
  40 di spazio nella memoria */
  int v[40];

  /* scrivendo in questo modo, invece, inizializzi un array che 
  automaticamente alloca la memoria al numero di numeri
  (in questo caso 5) che scrivo dentro l'array */
  int v[]{ 5, 6, 4, 3, 5}

  /* quando scrivo questo, il numero equivale al primo valore 
  dell'array (in questo caso 5) */
  int v[0]; // ridarà 5 

  /* per inserire un input dentro un array la sitassi è cos'ì: */
  int v[20]; /* per poter inserire un valore nell'array bisogna 
  che lo spazio in cui memorizzi il valore esista */
  int v[1] = 10; 
```
